# Data

_Links, embeds, raw tables. The private dump for stuff you'll reference but don't need to write narrative around._

## Suggested sub-pages

- **Bookmarks** — saved links by topic
- **Embeds** — Loom recordings, Figma frames, Miro boards you reference
- **Raw tables** — CSV pastes, query results, anything tabular without a permanent DB home
- **Reference IDs** — Jira tickets watching, Slack thread bookmarks, etc.

## When to graduate

If a raw table here is queried more than 3x or referenced by team-visible work, convert it to a Notion database under `team_hub` and wire it into `notion_defaults.database_ids.*`.
