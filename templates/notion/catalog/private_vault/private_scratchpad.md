# Scratchpad

_Inbox for quick captures. Anything that doesn't have a home yet._

## How to use

- Add a child page per capture; date the title or add a date column if you convert this to a database later.
- Move pages out of here weekly — to `Notes`, `Documents`, `Tasks`, or trash.
- If something sits here >2 weeks unreviewed, that's a signal to either prioritize it or discard.

## Default post target

`notion_defaults.post_targets.scratchpad → private_scratchpad`

Skills that drop captures here:
- `/jstack:self diary`
- `/jstack:knowledge intake` (when source is informal)
- Quick voice transcripts (`elevenlabs-stt`, `deepgram-stt`)
