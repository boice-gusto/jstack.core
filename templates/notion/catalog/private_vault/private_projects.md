# Projects (private)

_Private project pages and scratch. Use this when a project is too early or sensitive for team-visible Tasks DB / Product Tasks._

## Use this for

- Skunkworks / pre-greenlight ideas
- Personal R&D
- Projects whose scope is still being shaped
- Anything that shouldn't surface in team boards yet

## Move to `team_hub` Tasks DB when

- The project has a stakeholder beyond yourself
- You want it tracked on the sprint board
- It needs Jira or roadmap visibility

## Conventions

- Each project is a child page with: goal, status, decision log, links, owner.
- When promoted, copy or move to `parent_pages.team_hub` and add a row to `database_ids.product_tasks` (or `tasks`).
