# Team context templates

Copy `*.md.example` to your private or team config directory (e.g. `./config/jstack-team/`) and remove `.example`. Point `jstack.config.json` → `team_context.base_path` and `team_context.files` at these paths.

See `skills/_core/references/integration-guide.md` for the `team_context` JSON shape.
